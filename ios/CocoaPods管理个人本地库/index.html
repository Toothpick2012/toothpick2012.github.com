CocoaPods管理个人本地库

@()[CocoaPods|技巧|pod] 有的时候我们的一些代码不是开源的，或者暂时不想放到Github上。我们只想使用CocoaPods引用本地的代码，接下来就来玩玩如何管理本地库

1、创建一个spec
pod spec create Peanut
2、spec配置

然后用文本编辑器或是vim来编辑

Pod::Spec.new do |s|
  s.name         = "TKVBaseProject"
  s.version      = "0.0.1"
  s.summary      = "A short description of TKVBaseProject."
  s.description  = <<-DESC
                   A longer description of TKVBaseProject in Markdown format.
                   DESC

  s.homepage     = "http://EXAMPLE/TKVBaseProject"
  s.license      = "MIT (example)"
  s.author             = { "qiumx" => "toothpick.v@gmail.com" }

  s.source       = { :git => "~/Desktop/Demo/TestLocalPod"}
  #本地库目录路径

  s.source_files  = "class/**/*.{h,m}"
  #源文件
  s.public_header_files = "class/**/*.h"
  #头文件
  s.xcconfig = { "HEADER_SEARCH_PATHS" => "$(PODS_ROOT)/TKVBaseProject" }
  #生成项目目录
  # s.dependency "JSONKit", "~> 1.4"
  #依赖库
end
3、Podfile配置

我们回到工程目录中，打开Podfile添加

pod 'TKVBaseProject', :path => '~/Desktop/Demo/TestLocalPod'

然后运行pod install
