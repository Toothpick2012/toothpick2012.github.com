Lightbox效果下的图库插件：Lightbox Gallery

   Lightbox效果是非常常见的图片显示效果，而这个插件将这个效果与图库（相册）整合到了一起。

   插件介绍：

   Lightbox相册插件能够将相册的浏览变为lightbox效果。

   将相册以Lightbox效果显示

   以工具栏方式显示图片说明

   显示与图片相关的]元数据

   将图库分配到几个页面中（分页功能）

   扩展默认图库选项

   选项页中设置<额外选项

   同时，你还可以将一般图片以lightbox效果显示。

   安装说明：

   1.编辑lightbox-gallery.js

   并根据你的设置检查第二行的路径。

   2.把lightbox-gallery文件夹复制到你的 directory into yourwp-content/plugins目录中

   3.从WordPress中的“插件”菜单中启用插件。

   在“images”文件夹中可能已经有图片了。用它们替换默认文件。

   搞定！：）

   如果你的主题目录中有“lightbox-gallery.css”和“lightbox-gallery.js”存在，Lightbox Gallery会调用它们。如果这两个文件不存在，插件会调用Lightbox Gallery里的默认“lightbox-gallery.css”和“lightbox-gallery.js”。这样你不必担心升]级插件时你创建的lightbox图库样式被覆盖。

   如何在不用[ gallery ]代码的情况下让一般图片在lightbox里显示？

   只需添加 rel="lightbox" 到 "a" 标签中即可。下面是一个例子：

   其他注意事项：

   如何使用

   这个插件的使用，简单来说就是添加 [ gallery ] 代码。此代码已在WordPress 2.5及更高版本上应用。 LightBox Gallery 插件自动将默认的图库查看方式转换为lightbox显示。 图片的说明文字以工具栏提示方式显示。 图片描述则在lightbox效果弹出后显示。

   高级设置

   用户可以通过下列额外的选项来扩展短标记 [ gallery ]的功能。

   lightboxsize

   此选项控制弹出的lightbox图片大小。默认是中等大小，但是你可以将它改为全屏。

   [ gallery lightboxsize="full" ]

   meta

   此选项控制是否显示EXIF信息。默认为不显示。如果你想显示照片信息，请将它设置为“true”。Lightbox的EXIF信息会显示照相机机型，光圈，焦距，快门速度以及拍摄时间等信息。

   [ gallery meta="true" ]

   class

   添加一个图库的 class 属性。默认为gallery1

   [ gallery class="gallery2" ]

   nofollow

   添加 rel="nofollow" 属性。默认为“false”。

   [ gallery nofollow="true" ]

   from, num

   定义从哪张图片开始显示以及显示多少张图片]。 如果图片数超过num , 就会显示导航条。你可以使用几乎和wp_link_pages

   一样的导航函数。

   [ gallery from="5" num="10" ]

   pagenavi

   如果你不想显示导航条，设置0。默认为1。

   [ gallery num="10" pagenavi="0" ]

   禁用插件

   完事 :)
