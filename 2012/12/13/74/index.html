关于彻底删除10.8.2深度休眠预留缓存的巨大垃圾文件

Dec 13 2012

       1.首先你要打开“文本编辑器"

     

&nbsp; &nbsp; &nbsp; &nbsp;2.在桌面上创建一个叫sleepimage的文档</span>

   

<span style="font-size:18px;"> &nbsp; &nbsp; &nbsp; &nbsp;3.然后打开你的Finder找到桌面选中这个文档回车，把它的后缀删掉 &nbsp; </span><span style="color:#333333;font-family:arial;font-size:18px;line-height:26px;background-color:#FFFFFF;">.rtf</span>

<span style="font-size:18px;"> &nbsp; &nbsp; &nbsp; &nbsp;4.新建一个Finder，按住SHIFT+COMMAND+G，然后输入前往的文件夹地址/private/var/vm。

&nbsp; &nbsp; &nbsp; &nbsp;5.在第3步打开的Finder的桌面上把sleepimage拖到/private/var/vm，也就是用文档编辑器创建的那个，系统会提示你巴拉巴拉巴拉的什么要求你输入管理员密码，输入后就替换了，记得是替换！不是保留两者！

&nbsp; &nbsp; &nbsp; &nbsp;6.然后选中你已经替换的sleepimage按COMMAND+i把&ldquo;已锁定&rdquo;打上钩钩。</span>

   

<span style="font-size:18px;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 当当当！！！大功告成啦！！！这个原理就是禁止系统修改此文件夹！然后那个巨恶心的深度睡眠预留空间就不会再被创建了！重启100+N次也不会了！是的！不会了！！！！

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span><span style="outline:none;color:#FF0000;font-size:18px;">此教程不会影响你正常的休眠和系统，请放心更改。
